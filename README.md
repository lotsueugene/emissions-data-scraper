<h2>Description</h2>
This project uses a Python-based pipeline to scrape global CO₂ emissions data from Worldometers, clean and store it in a SQLite database, and then analyze and visualize trends. It includes parsing raw HTML, managing local SQL data, and using statistical tools and visualizations to explore relationships between population and carbon emissions.
<br />

<h2>Languages and Utilities Used</h2>

- <b>BeautifulSoup</b><br />
- <b>pandas</b><br />
- <b>NumPy</b><br />
- <b>matplotlib</b><br />
- <b>SQLite</b><br />
